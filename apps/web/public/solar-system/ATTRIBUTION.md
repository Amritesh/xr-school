# Solar-system texture attribution

These locally optimized WebP files are educational presentation assets. They are not scientific-analysis products or live observations. Planet sizes, lighting, atmospheres, and some surface colours are intentionally adjusted for classroom legibility. The result is an **educational representation**.

- Venus — JPL/Caltech, Magellan/Venera/Pioneer composite: https://maps.jpl.nasa.gov/tmaps/pix/ven0ajj2.jpg

## Solar System Scope imagery-derived maps

Solar System Scope textures are based on NASA elevation and imagery data, with colour adjustment and gap filling. The texture set is published under CC BY 4.0; these copies were resized and converted to WebP for standalone-headset delivery.

- Sun: https://www.solarsystemscope.com/textures/download/2k_sun.jpg
- Jupiter: https://www.solarsystemscope.com/textures/download/2k_jupiter.jpg
- Saturn: https://www.solarsystemscope.com/textures/download/2k_saturn.jpg
- Neptune: https://www.solarsystemscope.com/textures/download/2k_neptune.jpg
